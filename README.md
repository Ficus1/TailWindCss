# 🎨 TailWindCss — Учебный лендинг на Tailwind CSS

Учебный проект для освоения Tailwind CSS. Вёрстка лендинга с использованием utility-first подхода без написания собственных CSS-классов.

---

## Стек технологий

| Компонент | Технология |
|-----------|-----------|
| Разметка | HTML5 |
| Стили | Tailwind CSS |

---

## Чему научился

- Utility-first подход к вёрстке
- Адаптивный дизайн через брейкпоинты (`sm:`, `md:`, `lg:`)
- Типографика, цвета, отступы и сетка через классы Tailwind
- Быстрое прототипирование интерфейсов без написания CSS

---
